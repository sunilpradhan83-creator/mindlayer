# Git Strategy

Project `.mindlayer/` is partially committed.

## Commit

```text
.mindlayer/project.md
.mindlayer/progress.md
.mindlayer/decisions.md
.mindlayer/context.md
.mindlayer/backlog.md
.mindlayer/risks.md
.mindlayer/index.md
```

These files preserve project intelligence for collaborators and future agents.

## Ignore

```text
.mindlayer/memory.md
.mindlayer/local.md
.mindlayer/private/
.mindlayer/sessions/
.mindlayer/cache/
.mindlayer/tmp/
```

These files may contain personal, private, generated, session, cache, or temporary memory.

Do not ignore the entire `.mindlayer` directory.
