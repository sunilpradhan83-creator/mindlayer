### Summary
MindLayer is human-approved, git-trackable memory for AI coding agents.

### Details
MindLayer helps solo AI-native developers using Codex, Claude Code, Cursor, or similar tools decide what to remember, where to store it, how to retrieve it efficiently, and how to keep memory safe, compact, transparent, and approval-gated.
